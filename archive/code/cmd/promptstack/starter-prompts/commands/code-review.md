Please review the following code for:

1. **Code Quality**: Is the code clean, readable, and well-structured?
2. **Bugs**: Are there any potential bugs or edge cases not handled?
3. **Best Practices**: Does the code follow language-specific best practices?
4. **Performance**: Are there any performance concerns or optimizations?
5. **Security**: Are there any security vulnerabilities?
6. **Documentation**: Is the code adequately documented?

Provide specific, actionable feedback with code examples where appropriate.
